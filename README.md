# Next App

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Prisma

Migration:

```bash
npx prisma migrate dev --name init
```

Prisma Studio:

```bash
npx prisma studio
```

Create data:

```bash
npx prisma db seed
```
